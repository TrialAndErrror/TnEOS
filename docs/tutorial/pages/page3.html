Config Menu
Open with Super + Shift + C
The config menu is a quick-access rofi launcher for your most-edited dotfiles. Invoke it from anywhere and pick from Awesome, ZShell, Neovim, GhosTTY, Zellij, SSH config, or Rofi — and the selected file opens straight in your editor.

Config files live under ~/.config/ and edits take effect the next time the relevant program is restarted. For Awesome, the config menu also triggers awesome.restart() automatically after saving.
