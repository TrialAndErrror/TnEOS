Project Picker
Open with Super + V
The project picker lets you quickly open up your projects in your favorite editor. It uses the global default editor to open up the folder that your repository is in, making it easy to quickly switch between projects.

The project picker searches for projects in the ~/repos folder, so make sure to clone your repositories there. It always adds ~/notes as an option as well so you can quickly open up your notes files and jot something down or look up a previous discussion.
