Remote Server Explorer
Open with Super + Shift + R
The Remote Server Explorer lets you quickly pick between remote servers and open them inside th default terminal, GhosTTY.

The Remote Server Explorer searches for servers configured in ~/.ssh/config and lets you pick between them. Check out the manpages for the SSH Config file with man ssh or on the web for more information on how to set up the config file.
