Display Picker
Open with Super + Shift + W
The display picker lets you instantly swap between saved monitor layouts. It reads shell scripts from ~/.screenlayout/ — each script typically calls xrandr with a pre-set arrangement — and presents them in a rofi menu.

Add your own layouts by setting up your display layout in arandr and then using the "Save" feature; by default, the layouts will be saved as *.sh files in ~/.screenlayout/. Useful for switching between laptop-only, docked, or presentation setups without touching a terminal.
