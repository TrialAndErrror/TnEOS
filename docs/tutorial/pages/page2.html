AwesomeWM Commands
This configuration includes all the basic AwesomeWM commands for moving between windows and workspaces. See below for a sampling of the more common keybindings from AwesomeWM, or check out the manpage with man awesome or the AwesomeWM Docs for more details on the default keybindings and how to modify them.

Windows

Super + FToggle fullscreen
Super + CClose window
Super + MMaximize / restore
Super + NMinimize
Super + J / KFocus next / prev
Super + SpaceToggle Floating

Workspaces

Super + 1–9Switch workspace
Super + ← / →Prev / next tag
Super + TabLast focused
Super + UOpen flashing tag
Super + SHotkeys popup
